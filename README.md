ONE EARTH – Sustainability Platform
A Django-based web app for promoting eco-friendly services and communication.

Features
- User login & authentication
- Contact form with email sending
- Responsive frontend UI
- Dynamic Django backend
- Deployed on Render
  
Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Django (Python)
- Database: SQLite
- Deployment: Render
